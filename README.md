# BestRunner

Запуск сервера:
  1. Перейти в папку backend
  2. Установить зависимости командой yarn install
  3. Запустить проект командой yarn start

Запуск клиента:
  1. Перейти в папку client
  2. Установить зависимости командой yarn install
  3. Запустить проект командой yarn start
